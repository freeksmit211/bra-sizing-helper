Sassy Lingerie Bra Size Calculator - Embed Instructions

This page demonstrates how to embed the Sassy Lingerie Bra Size Calculator into your website or online store.

There are two methods for embedding:

  1. Iframe embed (simplest)
  2. JavaScript embed (more customizable)

Method 1: Iframe Embed

This is the simplest way to embed the calculator. Just copy and paste this code where you want the calculator to appear:

<iframe src="https://your-domain-here.com/calculator" width="100%" height="800px" style="border: none; max-width: 100%;" title="Bra Size Calculator"></iframe>

Preview:

The iframe will load the calculator here.

Method 2: JavaScript Embed

For more customization options, you can use this JavaScript code:

<div id="sassy-calculator"></div>
<script src="https://your-domain-here.com/calculator-embed.js"></script>
<script>
  SassyCalculator.init({
    container: 'sassy-calculator',
    theme: 'light',
    primaryColor: '#f472b6'
  });
</script>

Preview:

The JavaScript embed will load the calculator here.

Additional Instructions

Once you've published your app, you'll need to:

  1. Replace your-domain-here.com with your actual domain where the calculator is hosted
  2. Adjust the height of the iframe if needed
  3. Customize the theme and colors for the JavaScript embed if using that method
